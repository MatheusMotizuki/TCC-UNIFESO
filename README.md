# informações

Este repositório contém o código-fonte e os arquivos do Trabalho de Conclusão de Curso (TCC) intitulado **"photoGraph: Modelagem de Fluxos de Edição Fotográfica Através de Estruturas de Grafos"**, desenvolvido para o curso de Bacharelado em Ciência da Computação no Centro Universitário Serra dos Órgãos (UNIFESO).

## Estrutura do Repositório

- `main.tex`: Arquivo principal do projeto LaTeX.
- `src/`: Capítulos e seções do trabalho em arquivos `.tex` separados.
- `bibliografia.bib`: Arquivo de referências bibliográficas.
- Outros arquivos auxiliares para apêndices, anexos e elementos pré/pós-textuais.

## Como compilar?
> caso esteja no windows.

1. Certifique-se de ter o [MikTeX](https://miktex.org/) instalado.
2. Compile o arquivo `main.tex` utilizando seu editor LaTeX favorito.

> caso esteja no codespaces leia [RUN.md](https://github.com/MatheusMotizuki/TCC-UNIFESO/blob/master/README.md)

## Sobre

Este trabalho propõe uma abordagem baseada em grafos para modelar fluxos de edição fotográfica, facilitando a visualização e automação de processos em softwares de edição.
